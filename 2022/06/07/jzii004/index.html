Introduce

题干

给你一个整数数组 nums ，除某个元素仅出现 一次 外，其余每个元素都恰出现 三次 。请你找出并返回那个只出现了一次的元素。

src：https://leetcode.cn/problems/WGki4K/

例子：

注意
  • 1 <= nums.length <= 3 * 104
  • -231<= nums[i] <= 231 - 1
  • nums 中，除某个元素仅出现 一次 外，其余每个元素都恰出现 三次

进阶：你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

Solutions

Solution 1 (数学方法，set)

nums 除了目标元素只出现 1 次，剩余的每个元素都出现 3 次，那么可以利用 set 去重来做题。

Solution 3 (bit wise, bit shifting)

不占用额外的空间。

二进制的数，比如 1001，出现 3 次，若记录 1 出现的次数，为 3003，之后对其进行 mod 3 操作，就没有余数。

Work & Thought

根据题目，-231 <= nums[i] <= 231- 1，我们可以创建一个 32 位长度的数组 cnt[]，用来记录所有数值的每一位共出现了多少次 1，之后对 cnt[] 数组的每一位进行 mod 3 操作，就可以得到只出现 1 次的值。

Afterthought

运算符的优先级
>> ， & >> 优先于 & ，所以(cur >> i) & 1不加括号也对
% ，& % 优先于 &，也就是乘除优先于逻辑
<< ， += << 优先于 += ，也就是位优先于赋值，所以ans += (1 << i)不加括号也对
  • 数每一位出现1的次数，可用移位cur >> i == 1 来实现；

time complexity: O(n)，遍历 nums 数组；

space complexity: O(1)，常数的空间申请；
